# Бот для онлайн игры Perfect World
## Как работает:
### открывается окно tkinter где с помощью кнопки запускается поиск всех мобов в округе - составляется список
### выбираются нужные мобы , безопасные точки для отхила, бафов, медитации
### запускается кнопкой
#### с помощью библиотеки распознавания текста resseract производится мониторинг экрана и обработки необходимой информации на интерфейсе , такие как координаты, здоровье, мана, название мобов, итемы
## Что может бот? :
### Отфильтровывать нужных мобов (бить скилами)
### Убегать в случае опасности (недостатка жизней)
### Подбирать лут
### Использовать зелья
### Призывать питомца , хилить его
### Бафаться по таймингу
### Фильтровать лут и поднимать нужный
### При опасности подниматься вверх , призывать полет , лететь на безопасную точку , отхиливаться , пробафываться , возвращаться на поле боя
